📊 Import Formulation from Excel
Upload an Excel file (.xlsx / .xls / .csv). The app will read each sheet as a separate formulation — one sheet per formula. Each sheet needs columns for ingredient name, amount, and optionally role and cost.
📂
Drop Excel file here or click to browse
Supports .xlsx, .xls, .csv — multiple sheets supported
📋 Expected Excel Format
Column headers the app recognises (any order, case-insensitive):
Ingredient / Raw Material / Material / Name — ingredient name (required)
Amount / Qty / Quantity / Parts / % — amount value (required)
Unit — unit of measure (optional, defaults to %)
Role / Function / Type / Category — functional role (optional)
Cost / Price / Rate / Cost/kg / AED — cost per unit (optional)
Sheet name = Formulation name. Works with your existing notebooks format.
